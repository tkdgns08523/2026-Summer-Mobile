03 · The Pivots

원래 이렇게 하려다, 이렇게 바뀐 지점들

기획과 갈라진 핵심 지점 세 가지.

🎩 캐릭터 꾸미기 — 아이템 장착에서 배경색으로

CASE 01
원래 계획
모자 · 선글라스 등을 이모지로 펫 위에 장착
→
실제 구현
원형 배경을 단색 5종(하늘·코랄·민트·레몬·라벤더)으로 교체
📷비교 스크린샷 자리
왜 바뀌었나
  • 펫·성장 단계마다 이모지 구도가 제각각(정면 클로즈업 vs 옆모습 전신)이라 액세서리 좌표가 계속 어긋남
  • 이모지는 위치 정보 없는 완성된 그림이라 근본적 한계 — 배경색으로 단순화

🏆 랭킹 친구 추가 — 닉네임 검색에서 로컬 목업으로

CASE 02
원래 계획
닉네임 검색으로 다른 사용자를 친구 추가
→
실제 구현
로컬 계정 + 목업 데이터로 랭킹 구성
📷랭킹 화면 스크린샷 자리
왜 바뀌었나
  • 서버 인증이 없는 로컬 전용 앱이라 검색할 대상(다른 사용자 데이터) 자체가 없음
  • 추가 예정 기능 — 서버 연동 전까지 의도적 보류

📈 통계 주간 그래프 — 꺾은선에서 막대로

CASE 03
원래 계획
일간 막대, 주간 꺾은선(라인) 그래프
→
실제 구현
일간·주간 모두 막대 그래프(Canvas 자체 제작)
📷통계 화면 스크린샷 자리
왜 바뀌었나
  • 외부 라이브러리 없이 Canvas 뷰 BarChartView 하나만 만들어 재사용
  • 라인 차트를 별도 구현하지 않아 주간도 막대로 표시

06 · Summary

정리

10개 모듈 전부 정상 작동. 기획과 달라진 지점은 세 가지.

CASE 01

캐릭터 꾸미기

아이템 장착 → 배경 색상. 이모지 좌표 정렬의 근본적 한계 때문.

CASE 02

랭킹 친구 추가

닉네임 검색 → 로컬 계정 + 목업. 서버 없는 구조상 검색 대상이 없음.

CASE 03

주간 통계 그래프

꺾은선 → 막대. 자체 제작 차트 뷰 하나로 통일했기 때문.
