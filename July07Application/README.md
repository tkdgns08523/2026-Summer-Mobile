# 2026-07-07 수업 내용 정리

1. 새 프로젝트(July07Application)를 생성하고 MainActivity, SecondActivity, ThridActivity, FourthActivity 4개의 화면을 구성했다.
2. `Intent`를 이용해 버튼 클릭 시 다른 Activity로 전환하는 화면 이동 기능을 구현했다.
3. `Intent.FLAG_ACTIVITY_CLEAR_TOP`과 `finish()`를 활용해 액티비티 스택을 정리하며 종료하는 방법을 익혔다.
4. `Toast` 메시지를 통해 사용자에게 화면 종료 등의 이벤트를 알리는 방법을 학습했다.
5. AndroidManifest.xml에 각 Activity를 등록하고, MainActivity를 런처(LAUNCHER) 액티비티로 지정하는 방법을 익혔다.
